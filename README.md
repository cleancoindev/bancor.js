Bancor 计算JavaScript版本
===

## Command 
```
fibjs index.js
```

## Usage
修改`index.js`中的 bancor 实例的入参，使用`bancor.[func]`输入买卖、解锁等行为，最后使用`bancor.printn()`或`bancor.printw()`打印结果。
